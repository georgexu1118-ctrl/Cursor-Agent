# Agent Instructions

This repository is designed to be worked on by multiple coding agents.

Global rules:
- preserve existing behavior
- do not remove tests
- do not commit secrets
- prefer simple, maintainable code
- explain risky changes
- keep architecture modular

Use cases:
- Claude Code: main builder
- Codex: debugging, review, test fixing, second opinion
- Cursor: manual editing, file navigation, terminal usage
