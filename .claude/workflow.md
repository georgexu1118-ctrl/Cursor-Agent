# Claude Workflow

For every task:
1. Read README.md, AGENTS.md, and relevant source files.
2. Understand the current architecture before editing.
3. Make the smallest correct change.
4. Add or update tests.
5. Run formatting, linting, type checks, and tests.
6. Summarize:
   - files changed
   - what was implemented
   - how to test it
   - next recommended step
